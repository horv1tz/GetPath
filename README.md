# GetPath

```powershell
Set-ExecutionPolicy RemoteSigned
```

### Using Git
```powershell
git clone https://github.com/horvitzoff/GetPath.git
```
### Using Releases
Download the archive from the link: https://github.com/horvitzoff/GetPath/releases
